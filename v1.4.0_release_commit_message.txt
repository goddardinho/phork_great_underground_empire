MAJOR RELEASE: v1.4.0 - NPC Conversation System & Debug Infrastructure

=== CORE FEATURES RELEASED ===

🗣️ Complete NPC Conversation System:
- Four interactive NPC commands: talk, ask, greet, say
- Rich dialogue tree system with branching conversations
- NPCManager for centralized conversation state management
- Sample NPCs: Hermit (simple) and Oracle (complex dialogue tree)
- Location-based NPC interactions and room integration
- Topic-based response system (treasure, grues, locations, help)
- Speech broadcasting system for ambient interaction

🛠️ Advanced Debug Infrastructure:
- Integrated debug mode accessible via --debug flag
- Debug menu system with comprehensive testing commands
- Real-time NPC system testing with 'debug npc' command
- World and object debugging capabilities
- Complete system state inspection and validation

📁 Professional File Organization:

- Removed redundant debug_scripts/ directory
- Enhanced tests/README.md with comprehensive testing guide
- Streamlined developer workflow with centralized testing resources

=== USAGE ===

NPC Conversations (In-Game):
- Start game normally: python3 main.py
- talk hermit / ask oracle about treasure / greet hermit / say "hello"

Debug Mode (Development):
- Start debug mode: python3 main.py --debug  
- Use debug commands: debug npc, debug menu, debug world

Testing (Validation):
- Run NPC tests: python3 tests/test_npc_system.py
- Integration tests: python3 tests/test_npc_integration.py
